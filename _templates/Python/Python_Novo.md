# Python novo

## criar a venv

```terminal

python3 -m venv .venv
source .venv/bin/activate
```

## atualizar o pip

```terminal
python -m pip install --upgrade pip
```

## criar notebook

Cmd+Shift+P → Create: New Jupyter Notebook

## selecione o interpretador da .venv

Cmd+Shift+P → Python: Select Interpreter → escolha o da pasta .venv.
